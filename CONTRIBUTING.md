# Contributing

Contributions must preserve the locked MTAW baseline and stay small and
reviewable. Before opening a pull request:

1. Read `AGENTS.md` and the build specification.
2. Do not add secrets, personal material, evidence, VM artifacts, or large
   binaries.
3. Label incomplete controls accurately as `planned`, `not_implemented`,
   `not_tested`, or `not_validated`.
4. Run repository checks and state unavailable tools as `NOT TESTED`.
5. Document destructive operations and proposed baseline changes explicitly.

Do not add a license or alter licensing status without explicit author
approval.
