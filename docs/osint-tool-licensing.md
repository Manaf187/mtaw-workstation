# OSINT Tool Licensing

The Trace Labs Awesome OSINT list is licensed under MPL-2.0, but that license
does not apply to the linked tools. MTAW records each selected or deferred
tool's own license in `manifests/osint-core-tools.yaml` and
`manifests/osint-specialist-tools.yaml`.

Automatic core licensing records:

- FFmpeg: Ubuntu package copyright controls apply; compiled components may be
  under LGPL/GPL-family terms depending on enabled libraries.
- ExifTool: Ubuntu package copyright controls apply; ExifTool follows
  Perl-style Artistic/GPL terms.
- Translate Shell: Unlicense.

Deferred core candidates:

- Sherlock Project: MIT.
- GoWitness: GPL-3.0.

Specialist profile records:

- SpiderFoot: MIT.
- Maltego: commercial proprietary terms.
- PhoneInfoga: GPL-3.0.
- Instaloader: MIT.

Do not add a tool to automatic installation until its own license, package or
artifact verification mechanism, redistribution posture, and account/API-key
requirements are recorded. Do not use Trace Labs branding, logos, or language
that implies endorsement of MTAW.
